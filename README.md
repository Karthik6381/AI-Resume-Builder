
# AI Resume Builder

An interactive **AI Resume Builder web application** that allows users to create professional resumes easily by entering their details and selecting different templates with a live preview.

## 🚀 Project Overview

The AI Resume Builder helps users quickly generate well-structured resumes without formatting manually. Users can input personal information, education, experience, and skills, and instantly see a live preview of their resume.

This project was developed as part of an **Advanced Frontend Development course project**.

## 🛠 Tech Stack

* **Frontend:** React.js
* **Build Tool:** Vite
* **Styling:** CSS / Tailwind CSS
* **Icons:** Lucide React
* **Version Control:** Git & GitHub

## ✨ Features

* Resume template selection
* Live resume preview
* Editable personal information
* Education and experience sections
* Skills section
* Responsive user interface
* Clean and modern design

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/Karthik6381/AI-Resume-Builder.git
```

Go to the project directory:

```
cd AI-Resume-Builder/client
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Open in browser:

```
http://localhost:5173
```
